== Description ==
We have developed add-on for All in One SEO Pack plugin that enhances the current feature to next level.
So with the help of this add-on you can see the stats of meta Keywords /Description/Title.

= Features =
1. Very Simple Clean Code plugin to list the Keywords /Description/Title stats of All in One SEO plugin
2. Addon of All in One SEO Pack
3. List Keywords /Description/Title by pages/ posts .
4. Colored flags: green representing Keywords /Description/Title used for single page/post and red representing Keywords /Description/Title are used for multiple pages/posts.
5. Search Keywords /Description/Title using post type and Keywords /Description/Title.
6. Sort Keywords /Description/Title in ascending and descending order.
7. Pagination:10 rows per page.
8. No technical skills needed.

== Installation ==
1. Upload the folder "meta-stats-all-in-seo-pack-addon" to "/wp-content/plugins/"
2. Activate the plugin through the "Plugins" menu in WordPress
3. Go to Wp-admin-> Meta Stats All One SEO -> Keywords Stats ,here you will find list of Keywords stats of All in One SEO Pack
4. Go to Wp-admin-> Meta Stats All One SEO -> Descriptions Stats,here you will find list of Descriptions stats of All in One SEO Pack
5. Go to Wp-admin-> Meta Stats All One SEO -> Titles Stats,here you will find list of Titles stats of All in One SEO Pack
