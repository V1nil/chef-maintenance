== Description ==
We have developed add-on for All in One SEO Pack plugin that enhances the current feature to next level.
So with the help of this add-on you can see the stats of Meta Description.

= Features =
1. Very Simple Clean Code plugin to list the Meta Description stats of yoast SEO plugin
2. Addon of All in One SEO Pack
3. List Meta Description by pages/ posts .
4. Colored flags: green representing Meta Description used for single page/post and red representing Meta Description are used for multiple pages/posts.
5. Search Meta Description using post type and Meta Description.
6. Sort Meta Description in ascending and descending order.
7. Pagination:10 rows per page.
8. No technical skills needed.

== Installation ==
1. Upload the folder "meta-description-stats-all-in-one-seo-pack-addon" to "/wp-content/plugins/"
2. Activate the plugin through the "Plugins" menu in WordPress
3. Go to Wp-admin-> All in One SEO Meta Description ,here you will find list of Keywords stats of All in One SEO Pack
